🟦 Internee.pk Clone – My Internship Task

This project is a clone of the Internee.pk website, created as part of my internship.
The goal was to replicate the layout, UI design, and basic functionality to improve my skills in web development and understand how real-world websites are structured.

🚀 About the Project

The Internee.pk clone is a frontend website built to look and feel like the original platform.
During this task, I focused on:

Clean and responsive UI

Reusable components

Mobile-first design

Modern React + Tailwind workflow

Smooth user experience

This project helped me improve my understanding of layout, component architecture, and styling.
